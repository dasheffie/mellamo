## 2026-02-03 - Complete Build

### Core App (US-001 to US-020)
- Express + SQLite + UUID stack
- 511 baby names (204 boy, 222 girl, 85 neutral)
- Diverse origins: Celtic, Scandinavian, Slavic, African, Asian, Middle Eastern, Latin American
- Swipe cards with drag/touch/keyboard support
- Session persistence via localStorage
- Favorites and rejects management

### ML Features (US-012, US-013)
- 41-dimension feature vectors:
  - Ending phoneme (16 one-hot)
  - Starting phoneme class (6 one-hot)
  - Syllable count (normalized)
  - Stress pattern (7 one-hot)
  - Sonorant ratio, vowel ratio
  - Gender score
  - Origin cluster (8 one-hot)
- Online logistic regression with SGD
- L2 regularization (0.01), learning rate (0.1)
- Name similarity matrix (cosine on phonetic vectors)
- Model weights persist in session
